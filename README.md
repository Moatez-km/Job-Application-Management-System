# Job Application Management System (JAMS)

A full-stack web application to manage job applications, online tests, and candidate evaluation.

## Tech Stack
### Backend
- Java 17
- Spring Boot
- Spring Security (JWT)
- Spring Data JPA
- PostgreSQL

### Frontend
- React
- Axios
- React Router

## Features
- User authentication (Admin / Candidate)
- Job application submission
- Online tests
- Application status tracking

## Architecture
- Controller layer
- Service layer
- Repository layer
- DTO-based communication
